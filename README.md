# Animated Wallpapers

A collection of animated wallpapers — code-generated animations, video loops, and more.

## Structure

```
wallpapers/
├── code/        # HTML/CSS/JS canvas animations, shaders, generative art
├── video/       # Video loop files (.mp4, .webm)
└── gif/         # Animated GIFs
```

## Usage

Browse the folders and pick a wallpaper. Code-based wallpapers can be run in a browser or with a live wallpaper engine like [Lively Wallpaper](https://www.rocksdanister.com/lively/).
